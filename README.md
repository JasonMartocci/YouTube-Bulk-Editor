Martocci Mayhem YouTube Bulk Editor

Bulk edit YouTube videos with this open-source GUI by @JasonMartocci! Update titles, descriptions, tags, thumbnails, & more. Save time with previews & backups. Free on GitHub – star & credit me! Subscribe: youtube.com/@JasonMartocci #MartocciMayhem

Setup:
1. Install Python 3.8+ from python.org/downloads
2. Install dependencies: pip install google-api-python-client google-auth-oauthlib
3. Get YouTube API credentials from console.cloud.google.com/apis/library/youtube.googleapis.com
4. Place credentials.json in the script folder
5. Run: python youtube_bulk_edit.py
